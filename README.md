# Discord Clone
This is a functional Discord clone built to demonstrate core social networking features and secure user authentication. The project is currently a Work in Progress (WIP).

## Features
### Authentication System
The application includes a complete login and registration system. It ensures that user access is managed securely through a structured backend flow.

### Security and Data Protection
Data integrity is a priority in this project.

### Password Hashing: 
All user passwords are encrypted using industry-standard hashing algorithms before being stored in the database.

### Verification: 
The system performs secure comparisons during the login process to ensure that credentials match the stored hashes without ever exposing the plain-text passwords.

### Social Features
Friend System: Users can search for other members and send friend requests.

### Friend Management: 
Once a request is accepted, users are added to a dedicated friends list, enabling further interaction and status tracking.

## Status
This project is currently under active development. While the core authentication and social frameworks are functional, additional features and UI refinements are being implemented.

Detailed installation instructions will be provided as the project nears a stable release. Currently, the repository serves as a showcase of the backend logic and initial frontend integration.
